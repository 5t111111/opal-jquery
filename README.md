# RQuery

RQuery is a dom library for [http://opalrb.org](opal).

## Contributing

### Setting up

Clone the repo:

```
git://github.com/adambeynon/rquery.git
```

Install dependencies:

```
bundle install
```

### Running tests

Build dependencies:

```
rake dependencies
```

Build rquery library:

```
rake build
```

Build rquery specs:

```
rquery spec
```

Open `spec/index.html` and observe the output.
